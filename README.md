<p class=MsoNormal><span style='mso-fareast-font-family:"Times New Roman"'>#
bootcamp-challenge<o:p></o:p></span></p>

<p class=MsoNormal><span style='mso-fareast-font-family:"Times New Roman"'>This
is the Andela Ke Cohort 28 boot camp project for the maintenance/repairs
department tracker app<o:p></o:p></span></p>

<p class=MsoNormal><span style='mso-fareast-font-family:"Times New Roman"'><o:p>&nbsp;</o:p></span></p>

<h2><span style='mso-fareast-font-family:"Times New Roman"'>Description<o:p></o:p></span></h2>

<p class=MsoNormal><span style='mso-fareast-font-family:"Times New Roman"'><o:p>&nbsp;</o:p></span></p>

<p class=MsoNormal><span style='mso-fareast-font-family:"Times New Roman"'>I am
required to make a maintenance or repairs tracker app for the
maintenance/operations department. The users should be able to sign in if they already
have an account or sign up if they do not have one.The users should be able to
make requests as well as view their requests.<o:p></o:p></span></p>

<p class=MsoNormal><span style='mso-fareast-font-family:"Times New Roman"'>The
application should also allow for admins to log in and see each request that
has been made, approve or reject it and resolve the request<o:p></o:p></span></p>

<p class=MsoNormal><span style='mso-fareast-font-family:"Times New Roman"'><o:p>&nbsp;</o:p></span></p>

<p class=MsoNormal><span style='mso-fareast-font-family:"Times New Roman"'><o:p>&nbsp;</o:p></span></p>

<p class=MsoNormal><span style='mso-fareast-font-family:"Times New Roman"'>Postman

<h2><span style='mso-fareast-font-family:"Times New Roman"'><o:p>&nbsp;</o:p></span></h2>

<h2><span style='mso-fareast-font-family:"Times New Roman"'>Challenge 1: User
Interface<o:p></o:p></span></h2>

<p class=MsoNormal><span style='mso-fareast-font-family:"Times New Roman"'>After
carefully reading the instructions, we are required to first make pivotal tracker
stories and create a github repository initialized with a readme.md file and
clone it. Create a new develop branch off master and add a feature branch. In
the feature branch create a directory calling it UI where you will save all the
pages you have created. After completing with the features we merge the feature
branch with develop branch and create pull requests for our pivotal tracker
stories Design the front end user interface for the application, which will
outline the stipulated tasks. HTML, CSS and Javascript should be used for the
design and responsiveness of the interface. No pre-built frameworks such as
bootstrap should be used in the project.<o:p></o:p></span></p>

<p class=MsoNormal><span style='mso-fareast-font-family:"Times New Roman"'>Features
required are:<o:p></o:p></span></p>

<p class=MsoNormal><span style='mso-fareast-font-family:"Times New Roman"'><span
style='mso-spacerun:yes'>&nbsp;&nbsp;&nbsp; </span>1. Users can create an
account and log in.<o:p></o:p></span></p>

<p class=MsoNormal><span style='mso-fareast-font-family:"Times New Roman"'><span
style='mso-spacerun:yes'>&nbsp;&nbsp;&nbsp; </span>2. The users should be able
to make maintenance or repairs request.<o:p></o:p></span></p>

<p class=MsoNormal><span style='mso-fareast-font-family:"Times New Roman"'><span
style='mso-spacerun:yes'>&nbsp;&nbsp;&nbsp; </span>3. An admin should be able
to approve/reject a repair/maintenance request.<o:p></o:p></span></p>

<p class=MsoNormal><span style='mso-fareast-font-family:"Times New Roman"'><span
style='mso-spacerun:yes'>&nbsp;&nbsp;&nbsp; </span>4. The admin should be able
to mark request as resolved once it is done.<o:p></o:p></span></p>

<p class=MsoNormal><span style='mso-fareast-font-family:"Times New Roman"'><span
style='mso-spacerun:yes'>&nbsp;&nbsp;&nbsp; </span>5. The admin should be able
to view all maintenance/repairs requests on the application<o:p></o:p></span></p>

<p class=MsoNormal><span style='mso-fareast-font-family:"Times New Roman"'><span
style='mso-spacerun:yes'>&nbsp;&nbsp;&nbsp; </span>6. The admin should be able
to filter requests<o:p></o:p></span></p>

<p class=MsoNormal><span style='mso-fareast-font-family:"Times New Roman"'><span
style='mso-spacerun:yes'>&nbsp;&nbsp;&nbsp; </span>7. The user can view all
his/her requests<o:p></o:p></span></p>
